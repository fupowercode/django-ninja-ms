Bienvenido a la API de Usuarios

Para gestionar usuarios, puedes usar las siguientes rutas de la API:

  • Crear usuario: POST /api/usuarios/
  • Listar usuarios: GET /api/usuarios/
  • Actualizar usuario: PUT /api/usuarios/<usuario_id>/
  • Eliminar usuario: DELETE /api/usuarios/<usuario_id>/

Puedes acceder a la documentación completa de la API para más detalles.

Sitio de Administración

Para acceder al sitio de administración de Django, ve a /admin/.
